# Portfólio Pessoal - Gustavo Marques

Este é o repositório do meu portfólio pessoal desenvolvido com **HTML** e **CSS**, com foco na apresentação de minhas habilidades, trajetória e formas de contato. O projeto tem como objetivo destacar minha presença profissional na web e facilitar o acesso ao meu currículo e redes sociais.

## 🌐 Visualização

Você pode acessar o portfólio diretamente [clicando aqui](https://seu-link-aqui.com) *(substituir pelo link do GitHub Pages, Netlify, Vercel, etc.)*

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- Google Fonts
- Variáveis CSS (`:root`)
- Responsividade com Media Queries

---

## 🎯 Funcionalidades

- Navegação simples e direta entre seções: **Home**, **Sobre Mim** e **Currículo**
- Design responsivo para diferentes tamanhos de tela
- Destaque para links de redes sociais com hover interativo
- Imagem pessoal com borda decorativa
- Rodapé com assinatura personalizada
